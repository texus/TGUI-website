void tgui::BackendRendererSDL::setActiveRenderer ( SDL_Renderer * renderer)

Changes the SDL_Renderer object that is used to create textures.

This function is only needed when working with multiple windows. In such case each window has it's own SDL_Renderer and the correct one needs to be selected when textures are created.
