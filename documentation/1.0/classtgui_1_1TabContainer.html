Detailed Description

TabContainer widget.

A tab container widget is a control that allow user to switch between panels via tabs. If you are looking for something without holding panels then check out the Tabs class.

void tgui::TabContainer::setTabFixedSize ( float  fixedSize)

Changes the tab fixed size for fixed size alignment.

The tab fixed size is applied equally to all tabs of the TabContainer. A value less than or equal to 0.0f sets the width of the tabs to the automatic width (tabs fill the TabContainer). A value greater than zero sets the width of the tabs to the specified value.
