This function returns (0,0) is the height of the text next to the radio button is less than the radio button height. Otherwise (0, -offset) will be returned where the offset is the distance between the top of the text and the top of the radio button.

void tgui::RadioButton::setMaxWidth ( float maxWidth)

Sets the maximum width that is used to determine when to split the text over multiple lines.

Parameters
maxWidthWidth used for word wrapping
Warning
This width includes the size of the check box itself and the spacing between the box and the text. The max width thus starts calculating from the left position of the widget.

When set to 0 (default), there is no limit for the text width.
