◆ setTextPosition()

void tgui::ButtonBase::setTextPosition ( Vector2< AbsoluteOrRelativeValue >  position,
Vector2f  origin 
)

Changes the position of the caption text.

Parameters
positionPosition of the text within the button (position can be both relative or absolute)
originOrigin that determines which part of the text is placed at the given position

The position consists of x and y components which are either a constant (e.g. 5) or a percentage string (e.g. "5%"). The origin consists of x and y components with values between 0 (left/top) and 1 (right/bottom).

The default text position is centered in the button, i.e. ({"50%", "50%"}, {0.5f, 0.5f})

Example that places the left side of the text at 10% of the button width and places the vertical center of the text at 25px from the top of the button:

button->setTextPosition({"10%", 25}, {0, 0.5f});
